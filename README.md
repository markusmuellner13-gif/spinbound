# 🎰 SPINBOUND

**A roguelike slot machine.** Build a bag of synergizing symbols, spin the 5×4 grid, beat the ever-rising rent, and chase a high score that climbs forever. Inspired by *Cloverpit*, *Luck Be a Landlord*, and the scaling-combo rush of *Balatro*.

> ⚠️ **Play-money only.** Coins exist solely inside this game. They have **no real-world value**, cannot be bought, sold, cashed out, or transferred. There is no real-money gambling anywhere in Spinbound — it's a game of skill, luck, and combo-building.

## How to play

1. **Spin** — tap the button *or pull the lever* — fills the grid with random symbols drawn from *your bag*.
2. Symbols pay coins and **combo off their neighbours** (Multipliers ✖️ & Lightning ⚡ scale adjacent symbols, Midas 👑 buffs everything around it, Cats 🐱 eat Mice 🐭, three Sevens 7️⃣ explode, a Jackpot 🎰 row doubles the whole board…).
3. Every **5 spins** you owe **rent**. Make it or you're **evicted** — that ends the run.
4. Clear a floor → **draft a new symbol** into your bag, **🎲 reroll** the offer for coins, or skip for +5. Stack synergies into an unstoppable machine.
5. Rent climbs every floor. How deep can you go?

### Modes & feel
- **23 symbols** across 4 rarities (common → legendary) for deep, replayable synergy-building.
- **🗓 Daily Seed** — everyone gets the same symbols & shops each day; compete for the deepest floor.
- **Synthesized sound + haptics** (Web Audio, zero asset files) with a one-tap mute.
- **Auto-dismissing coach cards** teach the goal, rent timing, and combos as you play.
- Animated **slot-machine lever**, drifting neon background, juicy payout pops.

## Install as an app

Spinbound is a **PWA** — open it in your phone browser and choose **"Add to Home Screen."** It then runs fullscreen, offline-capable, just like a native app.

## Run locally

It's a static site — no build step.

```bash
npx serve .
# or
python -m http.server 8000
```

Then open the printed URL.

## Tech

- Vanilla HTML/CSS/JS — zero runtime dependencies
- Data-driven symbol engine (`symbols.js`) — every symbol is a small `onScore()` function, easy to extend
- PWA: `manifest.json` + `sw.js` (offline cache)
- Icons generated by `tools/generate-icons.js` (pure Node, no deps)
- Deployed on **Vercel**, auto-deploys on every push to `main`
